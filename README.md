# Sage Sudoku
Sage Sudoku is a minimalist Sudoku game built with the Unity Engine. 
The game focuses on core Sudoku functionality while providing features such as mistake tracking, timing, and persistent game state across sessions.

## Authors
**Laurel Latt**

## Features
- Timer and mistake counter
- Persistent storage for in-progress games and user settings across sessions

## Tech Stack
- Unity (version 6000.2.7f2)
- C#
- JetBrains Rider (IDE)
